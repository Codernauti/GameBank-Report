\begin{abstract}
%The abstract goes here.
Applications that interact with real world games are becoming more and more 
interesting to study, as they enhance the gameplay, making it more interactive 
and fun to play. Furthermore, multiplayer mobile games allow 
players to interact between each other. In this paper, we present GameBank, a 
pervasive game that distribute the burden of managing money in board games to 
the players, through a well defined and already existing wireless link: 
Bluetooth.
\end{abstract}
